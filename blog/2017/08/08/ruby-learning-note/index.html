安装（不走rvm）

  • Manjaro下用Pacman装 ruby，nodejs：
  • 改.bashrc
  • 安装rails（国内最好换源）
  • 国内换源（https://gems.ruby-china.org/）

初步使用

  • 安装barckets（-bin）

    yaourt barckets

  • 或安装sublime

    yaourt sublime-text / sudo pacman -S sublime-text

  • 或安装visual-code

    sudo pacman -S visual-studio-code

  • Rails

  • 编辑 controllers/home_controller.rb
  • 编辑 views/home/index.html.erb
  • 编辑 configs/routes.rb
  • Rails Server

禁用caffee

  • Terminal
  • 编辑Gemfile，注释caffee

  • 重新编辑 controllers/home_controller.rb，views/home/index.html.erb， routes可不动

  • Terminal

  • rails s

禁用turbolinks, 容易和前端架构冲突，如react，轻量化可开启

  • 编辑Gemfile，注释turbolinks
  • 重新编辑 app/assets/javascripts/application.js，删除
  • 重新编辑 app/views/layouts/application.html.erb，更改
  • Terminal
  • rails s

增加first.html.erb, second.html.erb, show_post.html.erb 在app/views/home/目录下

显示所有routes

需 cd 至 rubyrails 文件夹下运行， 如果在更子目录下会自动跳转至主目录

使用 Gem 安装 bootstrap 3

  • /Gemfile改版本号

  • 终端更新

  • rename application.css to application.scss

  • import Bootstrap style in app/assets/stylesheets/application.scss

  • 调整js app/assets/javascipts/application.js

*只用 sass 丢进 RoR

  • 下载sass

  • 在 app/assets 放 fonts, images; app/assets/javascript 放 js； app/assets/styelesheets放 css

  • 删掉 compile 的 js

  • 调整js app/assets/javascipts/application.js

  • rename application.css to application.scss

  • 调整app/assets/stylesheets/application.scss, remove all the *= require_self and *= require_tree .,
